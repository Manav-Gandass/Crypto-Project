# 🚀 Crypto Dashboard

A simple cryptocurrency dashboard built using **basic HTML, CSS, and JavaScript**. It displays real-time cryptocurrency prices and price charts, allowing users to search for coins and view market data in different currencies.

## Features

- View real-time cryptocurrency prices
- Search for cryptocurrencies
- Interactive price charts
- Automatic data refresh
- Currency selection (USD, INR, EUR)
- Responsive design
- Loading and error states
- Cached API responses for improved performance

## Technologies Used

- HTML
- CSS
- JavaScript
- CoinGecko API
- Chart.js
- Local Storage

## Project Structure

```
├── index.html
├── styles.css
├── script.js
└── README.md
```

## How It Works

- Loads a list of popular cryptocurrencies when the application starts.
- Displays the current price and 24-hour price change for each coin.
- Shows an interactive price chart for every cryptocurrency.
- Users can search for cryptocurrencies using the search bar.
- Prices can be viewed in USD, INR, or EUR.
- Data is automatically refreshed every 30 seconds.
- API responses are cached using Local Storage to reduce unnecessary requests.
